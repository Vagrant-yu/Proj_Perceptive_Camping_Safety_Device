# 项目方案要点 

## 一、项目定位与名称
* **项目名称**：智能环境感知的户外露营安全保障装置 
* **核心定位**：从“智能帐篷”升级为独立的安全预警设备，可配合帐篷使用，重点从“环境监测”转向“主动安全预警” 。

## 二、双赛区分策略

### 1. 合泰杯
* **核心限制**：必须以合泰单片机为主控，禁止使用性能更强的单片机 。
* **技术方案**：采用红外接收器进行360°扫描，检测恒温动物入侵并预警；对于冷血动物入侵，通过物理关紧帐篷进行防御 。

### 2. 嵌赛
* **核心优势**：无硬件限制，主打性能提升 。
* **技术方案**：采用红外摄像头进行360°监测，结合AI生物识别，实现更精准的入侵预警 。

## 三、技术模块与判断逻辑

### 硬件模块
* **保留模块**：测温、湿度、声音、光照等环境感知模块 。
* **新增模块**：
    * 红外俯仰加360°扫描模块 
    * 4G远程通信与物联网推送模块 
    * 声光预警模块（可选） 
    * 超声波探测模块（用于识别响尾蛇等特殊生物） 

### 逻辑判断

#### 1. 判断恒温动物入侵
* **融合方案**：采用声音传感器 + 红外传感器融合判断 。
* **声音**：检测动物活动产生的声响 。
* **红外**：检测恒温动物的热辐射信号 。
* **判定条件**：两者同时满足，判定为恒温动物入侵 。

#### 2. 判断下雨
* **融合方案**：采用声音传感器 + 红外传感器 + 湿度传感器融合判断 。
* **声音**：识别雨滴击打帐篷 / 设备产生的特征声响 。
* **红外**：感知降雨带来的环境温度变化 。
* **湿度**：检测环境湿度是否急剧升高 。
* **综合逻辑**：湿度骤升提供“环境变潮湿”的基础证据，红外温度下降佐证“低温雨水进入”，特定声音模式最终锁定“降雨”，避免单一传感器误判 。
* **判定条件**：三者同时满足，判定为下雨 。
